What is CSP?

Content Security Policy (CSP) is a security standard that helps prevent cross-site scripting (XSS), clickjacking, and other code injection attacks by controlling which resources the browser is allowed to load.

Current CSP Policy

  • default-src 'self': Only allow resources from the same origin
  • script-src 'self' 'nonce-xxx': Allow scripts from same origin and with valid nonce
  • style-src 'self' 'nonce-xxx': Allow styles from same origin and with valid nonce
  • img-src 'self' data: https: Allow images from same origin, data URLs, and HTTPS
  • frame-ancestors 'none': Prevent this page from being embedded in iframes

Note: Nonces are unique per request and provide better security than 'unsafe-inline'
